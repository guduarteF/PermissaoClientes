# CRUD Clientes e Permissões

## 📖  Descrição

Projeto Backend em C# .NET Blazor com EntityFramework e SQL Server para CRUD de clientes. 
A página exibe uma listagem com os clientes e permite adicionar , editar e remover estes.

- Editar Cliente
- Remover Cliente

- Bootstrap
